== Description ==

This plugin implements a payment gateway for Business Directory Plugin to let buyers pay with Bitcoin, Litecoin, Ripple, and other cryptocurrencies via CoinPayments.net.

== Installation ==

1. Upload files to the `/wp-content/plugins/wpbdp-gateway-coinpayments` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. In the Business Directory Plugin Settings page go to the Payment tab, then click CoinPayments.net.
4. Check "Enable CoinPayments.net" and enter your CoinPayments.net Client ID.
5. Select "Enable Webhooks" and enter your Client Secret to use CoinPayments.NET webhooks.
6. Click "Save changes" and the gateway will be active.
